How an AI agent builds a real website

1. Research & structure

The agent browses competitors, docs, and examples — similar to how ChatGPT or Gemini research — but directly in the browser.

2. Generate code

Super writes HTML, CSS, and JS locally, opening editors and terminals like a human developer.

3. Push to GitHub

The agent logs into GitHub, creates repos, commits files, and opens PRs with your approval.

4. Publish & iterate

Deploy to static hosting, preview changes, and reuse prior steps via the computer-use cache.
